# homework3 Password Generator

This is a homework assignment. In this homework assignment, I had to create an application that generates a random password based on criteria the user selected. I had to modify a provided starter code for this assignment.

For this assignment, I added javascript script to generate a random password. I had to use functions, loops, conditions and arrays to achieve the requirement.

## Link to deployed website:

https://roan1088.github.io/homework3/

## Usage

This is a password generator that generates a random password based on the criteria the user selects. Click the "Generate Password" button to begin. Enter the length for the password between 8 and 128 characters long. Select what type of characters to be used for the password. Once atleast one type is selected, the generated password should appear in the textbox.
